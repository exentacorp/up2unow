# UP2U — Event Agency Website

Современный веб-сайт для ивент-агентства UP2U (up2u.moscow) — организатора корпоративных и городских мероприятий.

## 🎯 О проекте

UP2U — профессиональное ивент-агентство, специализирующееся на:
- Корпоративных мероприятиях
- Городских праздниках
- Федеральных проектах
- Спортивных событиях
- Культурных программах
- Технологических решениях (VR/AR)

**Целевая аудитория:** HR-директора, владельцы бизнеса, организаторы мероприятий

## 🚀 Деплой на GitHub Pages

### Шаг 1: Создайте репозиторий на GitHub

Создайте репозиторий с именем `up2u` (или измените `basePath` в `next.config.ts`)

### Шаг 2: Настройте package.json

Укажите URL вашего репозитория в поле `homepage`:
```json
{
  "homepage": "https://ВАШ_НИКНЕЙМ.github.io/НАЗВАНИЕ_РЕПОЗИТОРИЯ",
  ...
}
```

### Шаг 3: Измените basePath (опционально)

Если ваш репозиторий называется не `up2u`, отредактируйте `next.config.ts`:
```typescript
basePath: isExport ? '/НАЗВАНИЕ_РЕПОЗИТОРИЯ' : '',
```

### Шаг 4: Деплой

```bash
npm run deploy
```

Эта команда:
1. Соберёт статическую версию сайта (`npm run build:static`)
2. Отправит файлы из папки `out/` в ветку `gh-pages`

### Шаг 5: Настройте GitHub Pages

1. Зайдите в репозиторий → Settings → Pages
2. В разделе Source выберите: **Deploy from a branch**
3. Выберите ветку `gh-pages` и папку `/(root)`
4. Нажмите Save

Сайт будет доступен по ссылке из `homepage` через 1-3 минуты.

## 📦 Архив для загрузки

Готовый архив: `github-pages.zip` (16 МБ)

## 🛠 Технологии

### Frontend
- **Next.js 16** — React-фреймворк с App Router
- **TypeScript 5** — типобезопасный JavaScript
- **Tailwind CSS 4** — утилитарный CSS-фреймворк
- **Framer Motion** — анимации и интерактивность
- **shadcn/ui** — компоненты интерфейса

### Формы и валидация
- **React Hook Form** — управление формами
- **Zod** — валидация данных
